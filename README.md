# FinMate 💰
A full-stack **personal finance management application** that helps users track expenses, manage budgets, save money, and split group expenses efficiently.

FinMate allows individuals and groups to monitor their financial activities, analyze spending patterns, and simplify shared expenses.

---

## 🚀 Features

### 🔐 Authentication
- User registration and login
- JWT-based authentication
- Secure password hashing
- Profile management

### 💳 Expense Management
- Add, edit, and delete expenses
- Filter by category, date, or month
- Expense analytics and summaries
- Monthly spending trends

### 📊 Budget Tracking
- Set monthly budgets by category
- Track spending against budget limits

### 👥 Group Expenses
- Create groups
- Add group members
- Record shared expenses
- Debt simplification algorithm for settling balances

### 🏦 Savings Vault
- Create savings goals
- Deposit and withdraw funds
- Track savings progress
